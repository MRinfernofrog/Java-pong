# Java-pong
Made pong in java with a very simple bot to play against and some keybinds that can change score, ball speed, and bot difficulty


keybinds:
q -   + ball speed
a -   - ball speed
e -   + bot speed
d-    - bot speed
r -      reset game
z and x to change the score
the bot is very simple just follows the ball at a slower speed, once you or the bot gets to 3 points you win or lose 

bugs:
faulty collisions when changing settings cuz of a bug i didn't see
